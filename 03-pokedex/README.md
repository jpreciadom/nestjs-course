# Execute in dev mode

1. Clone the repository
2. Install nest (id not installed)
```
$ npm i -g @nestjs/cli
```
3. Execute
```
$ npm ci
```
4. Run DB
```
$ docker-compose up -d
```
5. Start the app in watch mode:
```
$ npm run start:dev
```
